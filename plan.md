# 技术栈

| 层级             | 技术/工具                       | 选择理由                                         | 成熟度   |
| ---------------- | ------------------------------- | ------------------------------------------------ | -------- |
| LLM提供商        | OpenAI或Anthropic LLMs          | 复杂文档处理、财务计算、多步骤决策的顶级推理能力 | 生产就绪 |
| 代理编排         | LangChain与LangGraph            | 具备状态机、人机协同、并行执行的生产级工作流     | 生产就绪 |
| 后端运行时       | Python + FastAPI                | 异步性能优越，AI/ML库丰富，LangChain集成良好     | 生产就绪 |
| 文档AI (OCR)     | AWS Textract                    | 发票、采购订单、交付文件结构化数据提取           | 生产就绪 |
| 消息队列         | AWS SQS + SNS                   | 可靠异步任务处理，事件驱动触发，死信队列支持     | 生产就绪 |
| 主数据库         | PostgreSQL (AWS RDS)            | 支持ACID，存储代理记录、决策及审计日志           | 生产就绪 |
| 缓存/状态存储    | Redis (AWS ElastiCache)         | 代理状态缓存、会话管理、速率限制、短期记忆       | 生产就绪 |
| Connex集成       | REST API + Webhooks             | 事件驱动触发，API版本管理适配层                  | 现有平台 |
| 银行集成         | Connex银行API + OAuth2          | 利用现有融资集成层，扩展代理专用端点             | 现有平台 |
| 可观测性/LLM追踪 | LangSmith                       | 代理决策全追踪，成本监控，延迟跟踪，错误告警     | 生产就绪 |
| CI/CD与部署      | Jenkins, AWS ECS, Kubernetes    | 容器化部署，自动扩展，无停机发布                 | 生产就绪 |
| 前端（代理UI）   | React                           | 嵌入式仪表盘、人工干预界面，集成Connex UI        | 现有平台 |
| 提示管理         | LangSmith / 自定义提示库        | 版本控制，A/B测试，性能跟踪                      | 生产就绪 |
| 安全             | AWS KMS + Secrets Manager + WAF | 数据加密，密钥管理，API保护，PII数据掩码         | 生产就绪 |
